# scrapy_project

personal project of a future website that consists of a crawler that searches for data from Amazon and stores it in an SQL database.

## Technologies

- Python 3.9+
- Scrapy
- PostgreSQL

## Running the Spider

scrapy crawl amazonLink_spider -a url=""
